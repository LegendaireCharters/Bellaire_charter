# BellAir Broker Pro — Stage 1

PC-friendly GitHub Pages project.

## Upload
Upload all folders/files to your GitHub repo root:
- index.html
- manifest.json
- css/
- js/
- data/
- assets/

Then enable GitHub Pages from `main` / root.

## Stage 1 includes
- BellAir quote calculator
- Blank departure/destination by default
- Fixed iPhone bottom navigation
- Multi-stop route UI
- Basic fuel legality check per leg
- Editable aircraft settings
- Editable airport records
- Saved quote history
- WA map, switches to Australia outline when a non-WA airport is selected

## Next stages
Stage 2: load full public Australian airport CSV/database.
Stage 3: automatic fuel stop suggestions.
Stage 4: polished quote PDF and client fields.
